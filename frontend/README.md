# Gmail Expense Tracker - Frontend

React + TypeScript + Vite frontend for the Gmail Expense Tracker application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Start development server:
```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Dashboard with summary cards, monthly trends, and category breakdown
- Transaction list with filtering and pagination
- Settings page with sync history
- Dark mode support
- Responsive design
